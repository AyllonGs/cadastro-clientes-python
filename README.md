# Sistema de Cadastro de Clientes em Python

Projeto simples de sistema de cadastro de clientes desenvolvido em Python para prática de lógica de programação e manipulação de arquivos CSV.

## Funcionalidades

- Cadastro de clientes
- Sistema de login
- Armazenamento de dados em CSV
- Listagem de clientes cadastrados
- Sistema de menu no terminal

## Tecnologias utilizadas

- Python
- Biblioteca CSV (padrão do Python)

## Estrutura de dados

Os dados dos clientes são armazenados em um arquivo chamado:

clientes.csv

Campos armazenados:

- nome
- email
- telefone
- senha

## Como executar

1. Instale o Python
2. Baixe o arquivo `sistema_clientes.py`
3. Execute o comando:

python sistema_clientes.py

## Objetivo do projeto

Projeto desenvolvido para praticar:

- Estruturas de repetição
- Estruturas condicionais
- Funções
- Manipulação de arquivos
- Organização de código
